Detailed Description

template<typename T>
struct omni::Vec4< T >

Four-component vector with multiple named accessors.

The four components are accessible as:

  • x, y, z, w — homogeneous spatial coordinates
  • r, g, b, a — color channels with alpha
  • data[0..3] — raw indexed access
Note
normalize preserves w unchanged — it normalises only the xyz magnitude. This is intentional for homogeneous coordinates but may be surprising for pure 4D vectors.

TComponent type. Must support arithmetic operators and sqrt.
