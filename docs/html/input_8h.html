Detailed Description

Abstract input interface and keycode utilities.

Defines the Input base class for polling keyboard and mouse state and registering event callbacks, along with the full set of MED_KEY_*, MED_MOUSE_BUTTON_*, MED_GAMEPAD_*, and related input code constants.

Implement Input for a specific windowing backend (e.g. GLFW) by subclassing and providing the pure virtual methods. Use keycodeToChar to convert raw keycodes and modifier state into printable characters.
