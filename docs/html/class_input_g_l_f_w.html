Detailed Description

GLFW-backed input handler.

Installs GLFW callbacks on the native window obtained from Medium and forwards them into the Input callback system. Key and mouse button events fire both the per-(object, type) callback (via eventCallbackImpl) and the global callback (via globalCallbackImpl). Scroll events fire only the global callback with object code MED_MOUSE_SCROLL.

Note
Only one InputGLFW should be active per GLFWwindow at a time, since it overwrites the window's user pointer.
