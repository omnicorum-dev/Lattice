Detailed Description

Numeric constants for keyboard, mouse, gamepad, and joystick input.

Values mirror GLFW's defines and are safe to compare directly against GLFW callback parameters.

◆ keycodeToChar()

char keycodeToChar ( const int keycode,
const int mods )
inline

Converts a MED_KEY_* keycode and modifier state to a printable character.

Handles the full US-QWERTY layout including:

  • Letters (Shift XOR Caps Lock = uppercase)
  • Number row with shifted symbols (!, @, #, …)
  • Punctuation with shifted variants
  • Keypad keys (Num Lock assumed active)
Parameters
keycodeA MED_KEY_* constant for the pressed key.
modsActive modifier bitmask (see Modifier Flag Bits); only MED_MOD_SHIFT and MED_MOD_CAPS_LOCK are examined.
Returns
The printable char the key produces, or '\0' for non-printable keys (arrows, F-keys, etc.).

< Non-printable key (arrows, F-keys, modifiers, etc.).

inline

Registers a callback for a specific (object, type) input event.

Only one callback is stored per (eventObject, eventType) pair; registering again overwrites the previous callback.

Parameters
eventObjectThe input object (e.g. a MED_KEY_* or MED_MOUSE_BUTTON_* code).
eventTypeThe event type (MED_PRESS, MED_RELEASE, or MED_REPEAT).
callbackZero-argument callable invoked when the event fires.
