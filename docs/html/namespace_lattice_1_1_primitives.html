Detailed Description

Factory functions for primitive Model3D meshes.

All primitives are centered at the origin in object space. Use the owning entity's Transform to position, rotate, and scale them.

Generate a unit cube (side length 1, centered at origin).

6 faces, 12 triangles. Normals are per-face (flat shading). UVs are per-face, each face mapped to the full [0,1] range.
