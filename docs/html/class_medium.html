Detailed Description

Abstract platform window and rendering context.

Medium owns the native window lifetime and drives the main loop via mediumRun. Concrete subclasses provide platform-specific implementations of startup, shutdown, canvas rendering, cursor control, and file-system root paths.

Typical usage:

auto med = std::make_unique<MediumGLFW>(1280, 720, 320, 180, "My Game");
med->mediumStartup();
med->mediumRun([](f32 dt) {
// game update
});
med->mediumShutdown();
