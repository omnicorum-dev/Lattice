Detailed Description

An infinite ray defined by an origin and a normalized direction.

Used for mouse picking, line-of-sight checks, and collision queries. The direction must be normalized for intersection results to be correct.

Compute the shortest distance from the ray to a point.

Projects the point onto the ray and measures perpendicular distance. The projection is clamped to the ray origin (t >= 0), so points behind the origin return distance to the origin.

Test intersection with a sphere.

Solves the quadratic |origin + t*dir - center|^2 = radius^2. On a hit, t is the ray parameter of the nearest intersection in front of the origin. Retrieve the world-space hit position with at(t). Since direction is normalized, t also equals the distance to the hit.

if (ray.intersectsSphere(center, radius, t)) {
glm::vec3 hitPoint = ray.at(t);
float distance = t;
}
