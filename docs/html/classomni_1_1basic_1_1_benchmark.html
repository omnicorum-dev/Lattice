Detailed Description

Singleton profiler that writes a Chrome-trace-format JSON file.

Obtain the instance via Get, open a session with BeginSession, instrument code with PROFILE_FUNCTION or PROFILE_SCOPE, then close with EndSession. The resulting JSON can be loaded directly in chrome://tracing.

Note
Not thread-safe. Concurrent WriteProfile calls require external synchronisation.
