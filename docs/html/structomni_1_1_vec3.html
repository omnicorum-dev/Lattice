Detailed Description

template<typename T>
struct omni::Vec3< T >

Three-component vector with multiple named accessors.

The three components are accessible as:

  • x, y, z — spatial coordinates
  • r, g, b — color channels
  • L, R, C — left / right / center
  • u, v, w — texture coordinates
  • data[0..2] — raw indexed access
Template Parameters
TComponent type. Must support arithmetic operators and sqrt.
