Detailed Description

Structured, levelled logging to stderr and optionally a file (logPath).

Use the LOG_* macros rather than the _impl_log_* functions directly; the macros inject __FILE__ and __LINE__ automatically.

Individual levels can be silenced at compile time:

Preprocessor symbol Suppressed level
DISABLE_WARN LOG_LEVEL_WARN
DISABLE_INFO LOG_LEVEL_INFO
DISABLE_DEBUG LOG_LEVEL_DEBUG
DISABLE_TRACE LOG_LEVEL_TRACE

FATAL and ERROR are always emitted and cannot be disabled.
