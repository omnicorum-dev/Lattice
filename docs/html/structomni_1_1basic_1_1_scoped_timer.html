Detailed Description

RAII scope guard that logs elapsed time on destruction via LOG_INFO.

Lighter-weight than ScopedBenchmarkTimer — no session required, output goes straight to the log.

{
omni::basic::ScopedTimer t("my operation");
// ... do stuff ...
} // prints: "my operation took 3.142 ms"
